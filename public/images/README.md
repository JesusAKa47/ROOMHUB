# Imágenes del logo RoomHub

Coloca aquí tus imágenes para que se usen en toda la web:

| Archivo | Uso |
|--------|-----|
| **roomhub-logo.png** | Icono de las tres casitas. La página muestra este icono con el texto "RoomHub" al lado en la barra de navegación y en el pie de página. |
| **roomhub-logo-full.png** | Logo completo con el texto "TU ESPACIO IDEAL". Se muestra en la sección hero de la página de inicio. |

Si no existen estos archivos, la web mostrará el logo por defecto (SVG) y el texto "RoomHub" donde corresponda.

**Consejo:** Para que el logo en la barra azul se vea mejor, usa un PNG con **fondo transparente** en `roomhub-logo.png`. Si tu imagen tiene fondo blanco, la web aplica bordes redondeados y un borde suave para integrarla con la barra.
